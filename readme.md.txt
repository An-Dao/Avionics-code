Steps:

Download and install Arduino IDE 1.8.19 from https://www.arduino.cc/en/software
Download and install Teensyduino from https://www.pjrc.com/teensy/td_download.html
Clone the git repo and open flight_computer_template1.ino
Add the zip libraries in the libraries folder (Sketch->Include Library-> Add .ZIP Library)
Verify the code has no errors
Upload to Teensy 4.1 microcontroller
